## 2020-12-02   
永续合约是个大坑，资金费率很高，玩中长线趋势策略的慎入。  
昨天有人问我持仓这么久，资金费交了多少，我楞了一下，永续合约的资金费率和手续费一直以来我还真没怎么注意，我的交易频率很低，手续费是可以忽略的，资金费在我的认知里不是平台抽的，是多空互给的，那应该就是差不多能抵掉的吧，然后我就在那位老哥的指点下在okex的网页上导出了近3个月账单，筛选了资金费率统计了一下：-299u，居然不能抵掉，怎么说呢，这个数对我来说是有点没想到的，说它多吗，跟我这段时间盈利的5、6千u比也还不是不能接受的，但是万一我没盈利呢，或者后续行情震荡我开始亏损了呢，那这个费用是实打实交出去的成本呢。  
然后我就研究了一下这个资金费率  
8小时结算一次，最近一次0.043%，如果算成年化0.043%*3*365=46.085%，我不知道这样算对不对，反正我看着有点吓人。  
交易就是这样，要把所有的坑都踩一遍，以后不打算做永续合约了，准备转交割合约去，那个没有资金费。交割合约就是移仓比较麻烦，本来想着永续挺好，可以长期拿着，不用移仓，哎，还是老老实实交割吧。  
最后，关于这个账户的实盘展示的文章也到此为止，之前也想停了，没什么人看，现在要玩交割去了，停也就停了吧。多一事不如少一事，自己慢慢玩，嘿嘿！
